
# Arch Mate

A simple, user-friendly tool for managing and configuring Arch Linux systems. If it's installed just run 'amate' in your terminal!

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

```bash
  npm install
```

Start the server

```bash
  npm run start
```
